Space Invaders

Overview

I had the pleasure of completing my first independent project using Unity, a recognized and powerful game development tool. The purpose of the project was to explore the possibilities of developing 2D games in Unity, and it represented an exciting and educational journey for me. Along the way, I had a unique opportunity to explore the fundamental aspects of Unity, including the use of C#, a powerful programming language used within the Unity environment. By leveraging these tools, I experienced how the various elements in Unity can collaborate to create a well-structured and engaging 2D game.
